Program Sistem Penyewaan Buku Digital
Program ini adalah sistem penyewaan buku digital sederhana dengan dua jenis pengguna: Admin dan User.

Fitur Admin:

CRUD data buku (tambah, lihat, edit, hapus)

Fitur User:

Registrasi akun baru dengan saldo awal
Login dengan email, username, dan password
Melihat daftar buku tersedia
Menyewa buku dengan pemotongan saldo
Batas waktu sewa 7 hari
Cek saldo
