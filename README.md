Program Sistem Penyewaan Buku Digital
Program ini adalah sistem penyewaan buku digital dengan dua jenis pengguna: Admin dan User. Dimana seorang admin akan mengurus perbukuan untuk di sewa oleh user, 
sementara user dapat menyewa buku menggunakan saldo awal yang user masukkan saat membuat akun.

Fitur Admin:

CRUD data buku (tambah, lihat, edit, hapus)

Fitur User:

Registrasi akun baru dengan saldo awal\
Login dengan email, username, dan password\
Melihat daftar buku tersedia\
Menyewa buku dengan pemotongan saldo\
Batas waktu sewa 7 hari\
Cek saldo
